How to Use the Web Scraping Tool

This tool allows you to upload a .json file, extract specific product details, and download the results in an easy-to-use Excel format. Follow these simple steps:

Step 1: Prepare Your JSON File

- Ensure the file you want to upload is in JSON format.
- The JSON file should contain structured data, such as product names, prices, descriptions, or any other details you wish to extract.

Step 2: Open the Tool

- Visit the Tool page by navigating to the "Tool" menu option on the homepage.
- You will see an interface with an upload form.

Step 3: Upload Your JSON File

- Click on the "Choose File" button.
- Select your JSON file from your device.
- Ensure the correct file is selected (the file name should appear next to the upload button).

Step 4: Submit the File

- Once the file is selected, click the "Submit" button.
- The tool will start processing your file and extract the required data.

Step 5: Download the Excel File

- After the processing is complete, you will see a download link.
- Click the link to download the resulting Excel file to your device.
